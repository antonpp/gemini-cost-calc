# Reference Scripts

This directory contains the original Python scripts used for analyzing token usage and cost calculations.

## Contents

- **`generate_cost_table.py`**: A script that processes CSV data to generate cost breakdown tables, likely handling tiering and total costs.
- **`plot_with_threshold.py`**: A script that generates timeseries plots, potentially including threshold visualizations for cost or rate limits.

## Purpose

These scripts serve as a reference implementation for the logic and features built into the main web application. They are preserved here for historical context and verification purposes.
